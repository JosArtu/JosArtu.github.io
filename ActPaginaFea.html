1. HTML BÁSICO

¿Qué es HTML? Es el código base que estructura lo que ves en internet.

Head y Body:
- Head esconde los datos de la página.
- Body es el cuerpo, todo lo que escupe la pantalla está aquí adentro.

Etiquetas de estructura:

  • Section: Corta la página en diferentes secciones.
  • Article: Para meter un post o artículo independiente.
  • Aside: Para barras laterales con información extra.
  • Header: La cabecera.
  • Footer: Los pies de página.

2. CSS BÁSICO

¿Qué es CSS? Es lo que usamos para darle color y diseño a las páginas.

Propósito: Hacer que las páginas se vean bien.

¿Cómo aplicarlo al HTML? (3 formas)

  1. Archivo Externo: Haces un archivo .css aparte y lo enlazas.
  2. Interno: Pones una etiqueta style en el Head.
  3. En línea: Le pones el estilo directamente a la etiqueta

Selectores y Reglas:

¿Qué es un Selector? Es la herramienta para "apuntar" a qué cosa le quieres cambiar el color.

¿Qué es una regla CSS? Es la orden completa: elegir al objetivo y decirle qué estilo llevar.

  • Tag Selector: Le pega a todas las etiquetas iguales
  • Class Selector: Le pega solo a los que tengan un nombre de clase específico.
